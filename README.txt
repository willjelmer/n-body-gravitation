In this zip there are several .txt data files. Each of these dictates a starting state for each body in the system.
Run main.py and type which option you would like to see.

----------------
1: 2 body system
----------------

Simple 2 body system, forms a stable orbit

-------------------
2. Burrau's problem
-------------------

A 3 body problem solved analytically in the 1960s

---------------------------
3,4,5: Central solar system
---------------------------

Solar system model with first 4 planets - Mercury, Venus, Earth, Mars

-------------------------
6,7,8: Outer solar system
-------------------------

Solar system model with outer 4 planets - Jupiter, Saturn, Uranus, Neptune

---------------------------
9,10,11: Entire solar system
---------------------------

Entire solar system model

--------------------
12: Sun, Earth, Moon
--------------------

Scale model of sun, earth, moon system. Very hard to differentiate the earth and moon.

###############################################################################################

For each solar system model, there are two close encounter options:

Close encounter: An encounter with a 2.5 solar mass star at a distance of 500AU, the closest
density of stars observed in the universe

Extreme close encounter: An encounter with a 2.5 solar mass star that comes within the solar
system
